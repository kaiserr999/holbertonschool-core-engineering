# Exception Handling

This project covers how Python handles runtime errors and how to manage them using:

- `try` / `except`
- Specific exception types
- `else` and `finally` blocks
- Raising exceptions explicitly

## Tasks

### 0. Safe list printing
Function that safely prints `x` elements of a list using `try / except`.

### 5. Raise a message
Function that raises a `NameError` with a custom message.
