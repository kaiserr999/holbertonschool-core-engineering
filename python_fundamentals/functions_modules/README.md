# Functions and Modules

This directory contains exercises about defining and using functions, managing variable scope, and importing Python modules.

## Files
- `islower.py`: ASCII logic for lowercase character checking.
- `uppercase.py`: ASCII math for string uppercase transformation.
- `print_last_digit.py`: Computes and returns the last digit of an integer.
- `pow.py`: Computes the power of a number manually using loops.
- `simple_add.py`, `test_import.py`: Exercises explaining script execution and execution guards (`if __name__ == "__main__":`).
- `add.py`: Demonstrates proper module and function importing.
